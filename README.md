# cologne bot
Parts:
- ESP8266 MCU
- Kamoer KCM-ODM Pump
- IR sensor
- 3DP enclosure
- GLHF with wiring

Augments the stock controls on the pump unit using the ESP8266 and an IR sensor to start/stop the pump.
Configurable carpal-tunnel-less repackaging of bulk stock into smaller units for individual sale.
